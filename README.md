# Delivery-Service-BackEnd
Projeto WEB API DotNet 6.0 metodos https.

# Delivery Service

Api para gerenciamento e manutencao de rotas entre sistemas, 
o sistema possibilita a criação de servicos e rotas para o mesmos,
tambem disponibiliza a melhor rota entre dois pontos pre-definidos.

## Desenvolvimento da Webapi

Webapi desenvolvida em asp.net core, usando Mediatr para comunicação via mensagens,
Banco de dados SQL Server com entity framework e Swagger para mapeamento das rotas.

## Utilizados Metodos Https
- GET
- PUT
- POST
- DELETE.
